# Chibi Icons for Fabby AI

This folder contains chibi-style icons that represent different moods/states of the AI assistant.

## Required Icons

- `normal.png` - Default/happy state
- `thinking.png` - When processing or thinking
- `searching.png` - When searching for information
- `confused.png` - When confused or uncertain
- `sad.png` - When expressing sadness
- `crying.png` - When expressing strong sadness
- `slightly_annoyed.png` - When slightly annoyed
- `heart_giving.png` - When being helpful or caring

## Recommended Specifications

- **Size**: 256px × 256px (PNG with transparency)
- **Style**: Chibi/anime style
- **Format**: PNG with transparent background
- **Naming**: All lowercase with underscores

## Usage

These icons will be displayed in the AI chat interface based on the AI's current mood or the context of the conversation.
