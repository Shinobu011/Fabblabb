# ITO+ Image Assets

This directory contains image assets for the ITO+ (Intelligent Traffic Optimization) capstone project website.

## Directory Structure

### `/hero/`
Hero section background images (recommended: 1920x1080 or larger)
- `traffic1.jpg` - Main hero image showing traffic in Greater Cairo
- `traffic2.jpg` - Alternative hero image
- `traffic3.jpg` - Third hero image for slider

### `/system/`
System architecture and feature images
- `sensing.jpg` - Real-time sensing technology
- `ml.jpg` - Machine learning visualization
- `control.jpg` - Intelligent control systems
- `communication.jpg` - V2V and V2I communication

### `/features/`
Feature showcase images
- Add images showcasing key features of the system

### `/impact/`
Impact and results images
- Add images showing environmental, technological, and economic impact

## Image Recommendations

- **Format**: JPG or PNG
- **Resolution**: Minimum 1920x1080 for hero images, 1200x800 for feature images
- **Optimization**: Compress images for web to ensure fast loading
- **Theme**: Dark blue, traffic, technology, smart city themes

## Placeholder Images

Currently, the site uses gradient overlays as placeholders. Replace these with actual images for the best visual experience.
